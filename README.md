# Sales Forecasting using Machine Learning

##  Project Overview
This project predicts future sales using historical data and machine learning techniques.

##  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

##  Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Sales prediction model
- Visualization of trends

##  How to Run
1. Clone the repository
2. Install dependencies  
   `pip install -r requirements.txt`
3. Run the Jupyter Notebook

##  Output
Predicts future sales trends to support business decision-making.
